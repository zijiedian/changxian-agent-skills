# changxian-agent-skills

This repository contains the packaged Codex skills used by `changxian-agent` / `remote-control`.

## Included Skills

- `changxian-blog-publisher` - drafts and publishes `changxian-web` articles with Markdown, declarative poster cards, and JSON article records
- `changxian-remote-control` - captures host-bridge workflows plus bridge-backed memory, roles, and schedules

This skill is bundled into the bridge runtime and synced into the local Codex skills directory at startup.
